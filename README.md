# Zyntra 3D - Oficina de Manufatura Aditiva

Este é o e-commerce oficial da Zyntra 3D, especializado em impressão 3D de alta precisão, modelagem personalizada e venda de arquivos digitais.

## Funcionalidades
- Catálogo de modelos 3D físicos e digitais.
- Checkout seguro com Mercado Pago.
- Cálculo de frete dinâmico baseado em KM para Botucatu-SP.
- Agendamento de janelas de entrega.
- Painel Administrativo para gestão de produção e estoque.
- Assistente de IA (ZyntraBot) para suporte técnico.

## Tecnologia
- Next.js 15
- Firebase (Auth & Firestore)
- Mercado Pago SDK
- Genkit AI
- Tailwind CSS / Shadcn UI
